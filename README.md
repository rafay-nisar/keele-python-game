# Keele University Python Game Project

## Overview
This repository contains both **text-based** and **GUI-based** versions of a simple guessing game built in Python.

- **text_game.py** — Console version using standard input/output.
- **gui_game.py** — Graphical version using Tkinter.

## How to Run
1. Open the terminal.
2. Run the text version:
   ```bash
   python text_game.py
